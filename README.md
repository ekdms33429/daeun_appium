# Test Automation Using Appium and Python (Android)

### 환경 구성
  **Node.js**
  <br> **Appium Desktop**
  <br> **Java JDK 8**
  <br> **Android SDK**
  <br> **Python3**
  <br> **Appium-Python-Client(using pip)**
  <br> **PyCharm**

### Test Scenario
기존 고객의 Trip.com 앱 설치 후 최초 실행 > "My 계정" Tap > 이메일로 로그인 시도 및 성공
